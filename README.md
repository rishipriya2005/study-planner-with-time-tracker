# study-planner-with-time-tracker
Study planner project using python with tkinter as UI
Study Planner with Time Tracker (Python Tkinter)
📌 Overview
The Study Planner with Time Tracker is a desktop application developed using Python and Tkinter. It helps users organize their study schedule and track the time spent on each task, improving productivity and time management.
🎯 Features
Add, update, and delete study tasks
Track time spent on each task
Start/Stop timer functionality
User-friendly graphical interface
Lightweight and easy to use
🛠️ Technologies Used
Python
Tkinter (GUI Library)
▶️ How to Run the Project
Install Python (3.x recommended)
Clone this repository:
Bash
git clone https://github.com/your-username/study-planner.git
Navigate to the project folder:
Bash
cd study-planner
Run the program:
Bash
python main.py
📸 Output
Displays a simple GUI window
Allows users to manage study tasks
Tracks time dynamically
